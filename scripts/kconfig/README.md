# Kconfig (bbi86-flavour)

> [!WARNING]
> bbi86 kconfig system doesn't support all the bells and whistles of \
> the full-kconfig system, but only provides  what bbi86 needs.

### What's different?
Unlike most other kconfig-ports or kconfig systems, bbi86's kconfig build scripts is much easier to setup
and integrate into any GNU make project. But many of the original kconfig targets may have been changed or removed
to suit bbi86's use of it.
